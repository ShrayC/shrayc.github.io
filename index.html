Workflow

I tried to keep my workflow for this course simple. I usually used either kahn academy or atom as my text editor. Kahn academy is very useful because it lets you see your changes instantly and you can work a lot faster. I made commits to github manually because I find it easier, and if I use git bash or anything else to push commits then that's going to distract me from the actual work. When I did use atom, I just clicked on the html file to run the program because while it can be less convinient than having a server, it takes no time to set up. I also have multiple tabs open in google chrome such as stack overflow to find help for any problems I might be having, or github for version control.
